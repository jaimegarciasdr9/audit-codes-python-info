# Audit Internal Control Dashboard

This repository contains an **interactive audit dashboard template** used as the standard reporting format presented to the Audit Committee for the review and monitoring of continuous audit work.

> **[➜ Open Dashboard](audit-dashboard.html)**

---

## Overview

The attached file (`audit-dashboard.html`) represents the **standard reporting format** presented to the Audit Committee in the context of continuous auditing. Its purpose is to provide a comprehensive, structured, and up-to-date view of the status of ongoing and planned audit engagements.

This is a **reference template** with anonymised data, designed to be adapted to each specific engagement.

---

## Dashboard Sections

| Tab | Description |
|---|---|
| **Roadmap** | Gantt chart showing the timeline of all planned audit engagements |
| **Dashboard** | Global KPIs and status table for each audit (progress, result, responsible) |
| **Controls** | Detail of all controls executed per audit, including type, status and result |
| **Areas & Responsible** | Audit team organisation by area and assignment table |
| **Meetings** | Audit meeting calendar (kick-offs, progress reviews and closing meetings) |

---

## How to Use

To adapt this template to a real engagement:
1. Download the `audit-dashboard.html` file
2. Edit the data directly in the HTML (audits, controls, dates, responsible persons)
3. Open the file in any browser — no server or external dependencies required
